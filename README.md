# Credit-Card-Dashboard
🚀Thrilled to Share My Latest Project: Credit Card Financial Dashboard! 📊

I recently completed a Credit Card Financial Dashboard using Power BI, SQL, and Power Query to deliver real-time insights into key metrics and effectively monitor and analyze credit card operations.

Project Details:🔍

📌Extracted and analyzed transaction and customer data from a SQL database.

📌Built a dynamic and interactive dashboard in Power BI.

📌Streamlined data cleaning and processing to uncover actionable insights.

📌Delivered data-driven recommendations to support decision-making processes.

🔹 Key Insights:

✅ Customers aged 30-40 and 50-60 generate the highest revenue.

✅ Businessmen and self-employed customers contribute the most.

✅ High-income customers account for a major share of total revenue.

✅ Chip transactions drive more revenue compared to swipes and online payments.

✅ Q4 shows the highest revenue spikes due to increased spending.
